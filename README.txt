ENG - - - WheelCheckDataAggregator by Luca Furlano - - -

Before to start: use at your own risk!!!

1. place your .csv log file in this folder

2. edit 'config.json':

    2.1 set your .csv log file name in the field "input_file"
    
    2.2 set an aggregation order in the field "aggregation_order": 
        - a lower value makes your lut more similar to the original lut generation procedure
        - a higher value makes your lut smoother and more similar to your standard wheel behavior
        - recommended values: 2 - 10
        
3. run 'WheelCheckDataAggregator.jar'

DONE!

Note: the lut graph generated must be similar (at most smoothed) to that generated by the standard procedure. Pay attention.

ITA - - - WheelCheckDataAggregator by Luca Furlano - - -

Prima di iniziare: usare a vostro rischio e pericolo!!!

1. metti il tuo file di log .csv in questa cartella

2. modifica il file 'config.json':

    2.1 inserisci il mome del file di log nel campo "input_file"
    
    2.2 inserisci un ordine di aggregazione nel campo "aggregation_order":
        - un valore più basso renderà il lut file generato più simile alla normale procedura del lut generator
        - un valore più alto renderà il file di lut più morbido e simile al comportamento standard del tuo volante
        - valore raccomandati: 2 - 10
        
3. esegui 'WheelCheckDataAggregator.jar'

FATTO!

Nota: il grafico del lut generato deve essere simile (al più ammorbidito) a quello generato dalla procedura standard. Fare attenzione.

----------------------------------------

Example - Esempio (config.json):

{
	"input_file":"example.csv",
	"aggregation_order":5
}
